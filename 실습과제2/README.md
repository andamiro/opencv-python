# 자율주행과 로봇분야에서 Opencv가 어떻게 사용되는지 조사하라
- **주행 환경 및 객체 인식**: 카메라와 비전 AI를 통해 도로 위의 보행자, 다른 차량, 표지판, 교통 신호등의 위치와 종류를 실시간으로 감지하고 대응한다.
- **물류 및 제조 자동화**: 공장이나 물류 창고의 무인 지게차 및 로봇 팔이 비전 기술을 통해 부품이나 물건의 위치를 3차원으로 정밀하게 찾고, 표면의 흠집 유무를 판별한다.

# CLI 개발환경이 GUI 환경보다 장점을 조사하라
그래픽 요소를 렌더링할 필요가 없어 시스템 리소스를 적게 소모하며, 키보드 입력만으로 명령을 실행할 수 있어 작업 속도가 훨씬 빠르다.

# Numpy의 라이브러리가 파이썬 라이브러리 보다 빠른 이유를 설명하라
파이썬 리스트는 객체의 메모리 주소들이 힙(Heap)에 분산되어 있어 참조에 시간이 걸린다. 반면, Numpy 배열은 단일 데이터 타입의 값들이 메모리상에 연속적으로 나열되어 있어 CPU 캐시를 효율적으로 사용할 수 있다.
