# 다음 용어를 설명하라.
**단위행렬**은 주대각선(왼쪽 위에서 오른쪽 아래로)의 원소가 모두 1이고, 나머지 원소는 모두 0인 정사각 행렬이다.

**역행렬**은 정사각 행렬 A와 곱했을 때 단위행렬 I가 되게 하는 행렬이다.

**전치행렬**은 주대각 성분을 제외한 나머지 원소들의 행과 열을 바꾼 행렬이다.

# 파이썬 리스트와 넘파이 ndarray의 데이터 저장방식을 비교하고 넘파이가 처리속도가 빠른 이유를 설명하라
파이썬은 "동적 타이핑" 언어이므로 반복문 안에서 연산할 때마다 자료형을 매번 확인한다. 넘파이는 배열 생성 시 모두 같은 타입으로 고정되므로, 타입을 검사하는 아까운 연산 시간을 낭비하지 않는다.
