# CMOS 센서의 원리를 자세히 조사하시오.
반도체 중에는 MOS(Metal Oxide Semiconductor)라는 방식이 있다. 이는 반도체 중에 움직이는 자유전자가 튀어 나온 후 정공의 전하를 운반하는 방식의 반도체이다.
이 전하 운반에 자유전자와 정공 쌍방을 이용해 동작 속도를 고속화 시킨 것을 CMOS(Complementary Oxide SEmiconductor)라고 부른다.
각 화소에 빛이 조사돼 발생한 전하를 축적한 후 화소 자체에 신호로써 취급하여 증폭한다.

# 아날로그신호를 디지털신호로 변환하는 과정에서 샘플링과 양자화의 차이를 설명하시오.
**샘플링**은 시간의 흐름에 따라 신호를 일정한 간격으로 쪼개어 추출하는 과정이고, **양자화**는 그렇게 추출된 신호의 크기를 정해진 단계로 근사화하여 수치화하는 과정이다.

# 영상의 해상도의 정의와 이미지 센서의 개수의 관계를 설명하시오.
**영상의 해상도**란 화면을 구성하는 총 픽셀의 수로, 이미지 센서 내 픽셀의 개수가 많을수록 더 높은 해상도의 영상을 표현할 수 있다.

# 200X100 그레이스케일 영상 파일의 크기(바이트)를 계산하라.
총 픽셀 수는 100 x 200 = 20,000 픽셀이다.
그레이스케일 영상은 일반적으로 1 픽셀당 1 바이트를 사용하기 떄문에 20,000 x 1 바이트 = 20,000 바이트이다.

# 200X100 컬러 영상 파일의 크기(바이트)를 계산하라.
총 픽셀 수는 100 x 200 = 20,000 픽셀이다.
컬러 영상은 일반적으로 1 픽셀당 3 바이트를 사용하기 때문에 20,000 x 3 바이트 = 60,000 바이트이다.
